unix

Private repo to practise Unix APIs. 
- classic Unix APIs
- Linux specific APIs.

Includes :
- file IO
- epoll
- fork/exec
- process groups
- sockets
- CPU affinity
- signals
- shared memory
- message queue
- time
- deamons
- Process scheduling
- IO Scheduling
- IOCTL
- Resource Limits
- TTY
- brk
